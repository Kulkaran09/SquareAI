# SquareAI

SquareAI is a modern AI-focused business website built to showcase AI-powered services such as website creation, AI marketing content, and workflow automation for businesses.

The website is built using **Vite + React** and is designed to be fast, lightweight, and easy to deploy on platforms like **Netlify** with a custom domain.
